# Mario Pet Project

## Project Description

Simple Java project modeling pet characters (Luigi, Mario, Wario, Bowser). Choose a character to act as your pet and call behaviors to change stats.

## Run Instructions

How to run:

```
javac *.java
java Main
```


## Features

- Multiple pet character classes: `luigi`, `mario`, `wario`, `bowser`.
- Choose a pet character and call behaviors to modify stats (health, energy, happiness, scared level).
- Behaviors include finding mushrooms, exploring mansions, hitting question blocks, attacking Boos, and sleeping.
- Stats are clamped between 0 and 100.
